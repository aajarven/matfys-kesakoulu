Adjust view dialog

This dialog box allows you to enter the position and orientation of a viewport's camera numerically. It can be opened from the viewport menu.

Settings

View position

The coordinates of the virtual camera in space.

View direction

The direction vector of the virtual camera.

Projection type

When a perspective projection is used, the vertical field of view angle is specified in degrees. For a parallel projection, the vertical field of view is specified in units of length.
