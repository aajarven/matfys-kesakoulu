Show periodic images

Creates copies of all particles and bonds to visualize multiple images of a periodic system.

Parameters

Periodic images (X,Y,Z)

Enables the replication of the system in the each spatial direction. You can control how many times the system is copied in each direction.

Adjust simulation box size

If activated, the simulation cell size is increased to match the size of the extended system.

Assign unique particle IDs

If this option is activated, the modifier assignes new unique IDs to the particle copies generated by the modifier. Otherwise the duplicated particles will have the same identifiers as the original particles. This may cause problems with other modifiers (e.g. the Manual selection modifier), which rely on the uniqueness of particle identifiers.
