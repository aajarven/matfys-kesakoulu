Assign color

This modifier assigns a uniform color to all selected particles by setting their Color property. If no selection is defined, the color is assigned to all particles.

Note that OVITO uses a red color by default to highlight selected particles. Since this highlighting would mask the actual color assigned by this modifier, the modifier clears the current selection by default. If you need to preserve the selection for further use down the pipeline, you can suppress this behavior by checking the Keep selection option.

The color assigned by the modifier is an animatable parameter.
